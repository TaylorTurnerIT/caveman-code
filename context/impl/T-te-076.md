# T-te-076: Token-bounded LRU eviction preserving emitted message bytes
**Status:** done
`ToolResultCache({ tokenBudget: N })` triggers LRU sweep on `put`. Victims are sorted by `lastAccessed` ascending, filtered to exclude entries accessed in the current turn (T-077 invariant). Evicted entries are removed from the store and their token weight subtracted. Previously emitted message bytes (owned by the caller) are untouched.
Test: "LRU eviction respects token budget (T-076)".
Acceptance R5 AC-1, AC-2 met.
